# GLOBAL COLOR PALETTE REPLACEMENT GUIDE
# Old cyan/blue/purple scheme → New jade/bronze underground luxury scheme

## Main Gradients (3-color):
from-cyan-500 via-blue-500 to-purple-500 → from-[#2DD4BF] via-[#B8936D] to-[#14B8A6]
from-cyan-400 via-blue-400 to-purple-400 → from-[#2DD4BF] via-[#B8936D] to-[#1A4D4D]

from-cyan-400 to-blue-400 → from-[#2DD4BF] to-[#B8936D]
from-blue-400 to-purple-400 → from-[#B8936D] to-[#1A4D4D]
from-purple-400 to-pink-400 → from-[#1A4D4D] to-[#2DD4BF]
